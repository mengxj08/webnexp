Tutorial: #4 Decide blocks and trails:

#4.1: estimate the time for each trial (typically at least 3 trials per condition)

#4.2: estimate the time for each block

#4.3: balance the trials and blocks so that the main part of the experiment is within 45 minutes - 1 hour.

#4.4: combine with the condition arrangement

After setting correpsonding parameters (like No. of trails or blocks), NEXP will automatically calculate and estimate for you.

Definitions

Trial

  • A single repetition of a single condition/cell
  • A number of trials are used to increase reliability

Block

  • An entire section of the experiment
  • Repeated to analyze learning
  • Block has other definitions. NEXP uses this simplified definition for the purpose of the toolkit.

Example:


  • Block: same arrangement, repeated
  • Trials in each block: same content, with order randomized
