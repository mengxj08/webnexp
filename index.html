NEXP is a experimental design tool that facilitates design and running process of controlled experiments in the HCI field.

Create

Design

Design platform guides users to design an effective controlled experiment. The resulting design file is readable by running platform.

Running

Running platform provide a framework to read designed file and support users to conduct web-based experiments on participants with custimized implementations.

Analysis

Analysis platform enables resulting data to be rightly accessed by existing statistical analysis packages.

Workflow

1. Define the research question

2. Determine variables

3. Arrange conditions

4. Decide blocks and trials

5. Set instruction and procedures

Features

NEXP streamlines the experiment design workflow using a 5-step approach, provides automations and simulation to guide and speed up design process. NEXP also works as a HCI educational tool that presents tutorials and teach HCI beginners to design a controlled experiment.
