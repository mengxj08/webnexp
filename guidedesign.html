A guided 5-step approach to design controlled experiment: plenty of tutorials are provided during the design process and beginners are suggested to read related help information about each step.

Just hover your mouse whenever you are confused!

Start

#1 Define the research question

NEXP guides you go through five sub-steps in determining research question:
Step 1.1 Start with a general question
Step 1.2 Define target population
Step 1.3 Define task(s)
Step 1.4 Define measure(s)
Step 1.5 Define factor(s)

Step 1

#2 Determine variables

Based on your hypothesis, NEXP is able to suggest variables of the controlled experiment for you. In this step, you need to define independent variables and dependent variables for your experiment.
Notes:
Check how these factors affect the performance of these measures in a given task.
One key issue we need to consider is to eliminate the explanation of all possible factors so that the experiment can only be explained by the factors we manipulate.

Step 2

#3 Arrange conditions

From Independent Variables to Experimental Conditions:
In this step, you need to set the detailed arrangment for your experimental conditions.
You can set between-subject or within-subject for each independent variable. According to the levels of each independent variable, you can use different strategies (fully counterbalancing/Latin Square/No counterbalancing ) to reduce the learning effect, if any.

Step 3

#4 Decide blocks and trials

Step 4.1: estimate the time for each trial
Step 4.2: estimate the time for each block
Step 4.3: balance the trials and blocks so that the main time cost of the experiment is reasonable (usually within 1.5 hours)
Step 4.4: combine with the condition arrangement

Step 4

#5 Set instruction and procedures

Step 5.1: Recruit participants (determine target users and randomize)
Step 5.2: Consent form and pre-experiment questionnaire
Step 5.3: Instructions
Step 5.4: Practice trials
Step 5.5: Main experiment with breaks
Step 5.6: Post-experiment questionnaire and interview
Step 5.7: Debriefing
